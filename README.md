
Flask based smart pollution predictor powered by a pre-trained Scikit-Learn Random Forest Regressor.
Predicts PM 2.5 using 8 atmospheric parameters and stores each forecast log in Pandas generated Excel history with dynamic trend graph via Matplotlib.
This system is designed for air-quality awareness, forecasting, and environmental analytics research.

 8-Feature Atmospheric Input Prediction

 User Tracking (Name, IP, Timestamp)

 Persistent Logging in Excel (AQI History)

 Auto-Generated PM 2.5 Trend Graph

 Clean HTML Table Dashboard

 Responsive & Styled UI

 Downloadable Pollution History Excel

 Robust error handling for invalid input
 
Prediction Parameters

Mean Temperature	Avg atmospheric temp     -- : t
Maximum Temperature	Highest temp           -- : TM
Minimum Temperature	Lowest temp            -- : Tm
Sea Level Pressure	Pressure at sea level  -- : SLP
Humidity	% of moisture                    -- : H 
Visibility	Distance visible               -- : VV
Wind Speed	Current wind speed             -- : V
Max Wind Speed	Peak wind speed            -- : VM
